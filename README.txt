==================
django-micro-press
==================

django-micro-press is a pluggable app, intended to be embedded as a
simple newspaper in a game or other resource on a website.

Requirements
------------
Python >= 2.6
Django >= 1.2
django-template-utils (https://bitbucket.org/ubernostrum/django-template-utils)
lxml
django_11559_fix (git://github.com/oktoberliner/django_11559_fix.git)
    unless you are using Django trunk >= r16608.
