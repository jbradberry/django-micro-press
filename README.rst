==================
django-micro-press
==================

django-micro-press is a pluggable app, intended to be embedded as a
simple newspaper in a game or other resource on a website.

Requirements
------------
- Python 2.7, 3.5+
- Django >= 1.10, < 2
- template-utils (`https://bitbucket.org/ubernostrum/django-template-utils`)
- lxml
- jsonfield
